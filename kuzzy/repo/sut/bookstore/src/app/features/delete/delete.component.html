delete(id: string): Observable< ResponseStatus >

delete() method is used to removed a T object in database with a DELETE http request and returns a response with ResponseStatus type.

delete() method needs that your app respects the Genese standards :

* You must add a genese param to your model like this :

{{model | json}}

* The endpoint must wait a DELETE method.

* The api path must respect the REST api standards for a create() method, just like this :

http://my-path/book/id

In this example, the delete() method removes a book with the help of the NestJS backend.

Open your navigator console. In the tab "Network", you'll find the data received from DELETE request.
