getAll(params?: GetAllParams): Observable< T[] >

getAll() is used to get a list of objects from http GET request and to format them with T type.

getAll() method needs that your app respects the Genese standards :

* You must add a genese param to your model like this :

{{model | json}}

* The http request must be a GET method

* The api path must respect the REST api standards for a getAll() method, just like this :

http://my-path/book

In this example, the getAll() method receives the list of books sent by the NestJS backend.

Open your navigator console. In the tab "Network", you'll find the data received from GET request. In the tab "Console", you'll can compare these original data with the formatted objects sent as Observable by Genese. As you can see, Genese formatted all these objects with "Book" type.
