getArray(): Observable< ArrayResponse >

getArray() method is used to get a T object in database with a GET http request

This method needs that your app respects the Genese standards :

* You must add a genese param to your model to specify the endpoint's path :

* You must add a specific genese param "gnArrayResponse" to indicate to Genese that you're waiting an array of arrays :

* The endpoint must wait a GET method.

* The api path must respect the REST api standards for a GET method (example : http://my-path/array-of-arrays-of-books)

In this example, the getArray() method gets an array of array of books from the NestJS backend.

CAUTION :

The getArray() method doesn't return an ArrayOfArraysOfBooks object, but the content of the gnArrayResponse property, ie a Book[][] object.

Open your navigator console. In the tab "Network", you'll find the data received from GET request. In the tab "Console", you'll can compare these original data with the formatted objects sent as Observable by Genese. As you can see, Genese formatted the returned object with "Book[][]" type.
