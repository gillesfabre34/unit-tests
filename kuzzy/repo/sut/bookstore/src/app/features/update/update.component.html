update(id: string, updatedObject: T, options?: RequestOptions): Observable< T | any >

update() method is used to update a T object in database with a PUT http request and returns the updated object formatted with T type.

update() method needs that your app respects the Genese standards :

* You must add a genese param to your model like this :

{{model | json}}

* The endpoint must wait a PUT method.

* The api path must respect the REST api standards for a PUT method (example : http://my-path/book/id)

In this example, the update() method removes a book with the help of the NestJS backend.

Open your navigator console. In the tab "Network", you'll find the data received from PUT request.
